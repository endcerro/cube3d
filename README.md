# Cube3Dd

Simple pseudo game using raycasting and the MinilibX
The Minlibx is requiered to compile this project, macOS sources included
# Bonus features 

  - A spooky ghost
  - The sword of the legend
  - Adjustable FOV / MOVE SPEED / ROT SPEED
  - DARK MODE
